2.b

On utilise la commande suivante pour peupler le fichier qu-2.b.ods :

# pour les colonnes : 
ls qu-2.a/* | tr '\n' ';' | c
# pour le contenu de chaque colonne : 
function qu2 () {             
        echo $1
        bin/h-lettre < $1 | grep --regexp=' = [0-9]*' | cut -f3 -d ' ' | c
        read
}
for i in qu-2.a/*; do
  qu2 $i
  read
done

le c utilisé est un alias pour la commande zsh clipcopy, qui copie le contenue de l’entrée courante dans le clipboard.

Le premier fichier qu-2.b.ods contient un résultat non trié mais permettant la comparaison de tous les fichiers.

Ensuite on crée un fichier par texte :

function qu2 () {             
        echo $1
        bin/h-lettre < $1 | grep --regexp=' = [0-9]*' | tr -d ' ' | tr '=' ';' >> $1.csv
}
for i in qu-2.a/*; do
  file=$(basename $i)
  echo ";$file">qu-2.b/$file.csv
  qu2 $i
done
